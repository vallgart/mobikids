Стоимость подписки

2 400 сум / сутки

Платная подписка на сервис предоставляется на ежедневной основе. Оформление доступно через SMS, USSD-запрос или напрямую на web-портале https://mobikids.uz.

Активация

  • Web Автоматически при переходе на сайт портала.
  • SMS Отправьте текст START на короткий номер 0707
  • USSD Наберите команду *0707#

Деактивация

  • Web В личном кабинете через раздел «Подписка» на портале.
  • SMS Отправьте текст STOP на короткий номер 0707
  • USSD Наберите команду *0707*0#

Сервисные номера 0707 (SMS) и *0707# (USSD) предназначены для отправки информационных уведомлений и оперативного управления контент-услугой. Тарификация подписки производится с сервисных номеров 07000 (SMS) и 07001 (SMS).
